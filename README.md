# React Native Reanimated Sensor Visualisation Prototype

This is a showcase of the use of the react-native-reanimated librarys useAnimatedSensor function.
Using the 3 dimension values returned by the function I can manipulate UI objects, both size and position.
I've added a visual representation of sound levels using react-native-reanimated and sensors tab with live sensor data.

## Everything needed to get started

```bash
npx create-expo-app .
npm run reset-project
npx expo install react-native-reanimated react-native-worklets expo-sensors expo-audio
```
